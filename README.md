# Career Companion

Career Companion is a web-based project that helps job seekers evaluate job fit, identify missing skills and keywords, and manage job applications.

---
1) Download the project
On GitHub, click Code → Download ZIP
Extract the ZIP folder to your computer

2) Open in VS Code
Open VS Code
Click File → Open Folder
Select the extracted project folder

3) Open the terminal in VS Code
Go to Terminal → New Terminal

4) Install dependencies
In the terminal, run:
npm install

5) Start the project
Run:
npm run dev
